# Concórdia - Documento de Visão

> Concórdia — Sua comunidade não é um produto.

Este documento registra a visão institucional do projeto e complementa a documentação técnica principal.

## Proposito

Concórdia existe para oferecer uma alternativa ética de comunicação digital, com foco em privacidade, autonomia e transparência.

## Compromissos

- Projeto open-source e auditável.
- Self-host opcional para autonomia operacional.
- API aberta para bots e integrações.
- Governança orientada à comunidade.
