# API

Este diretorio define os contratos de comunicacao publica do Concordia.

## Principios

- API aberta e versionada.
- Compatibilidade documentada por versao.
- Sem lock-in para integracoes comunitarias.

## Conteudo esperado

- Especificacao de endpoints/eventos.
- Autenticacao e autorizacao.
- Limites de uso e politicas de erro.
- Guia para bots e webhooks.
