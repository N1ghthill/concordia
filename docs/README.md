# Documentacao

Este diretorio concentra a documentacao tecnica e operacional do Concordia.

## Estrutura

- `docs/architecture/`: arquitetura, decisoes e limites de sistema.
- `docs/api/`: contratos da API e diretrizes de integracao.
- `docs/security/`: modelo de seguranca, ameacas e mitigacoes.

## Regra de manutencao

Toda mudanca relevante de comportamento deve atualizar a documentacao correspondente no mesmo ciclo de revisao.
