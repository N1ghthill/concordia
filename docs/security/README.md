# Seguranca

Este diretorio organiza os artefatos tecnicos de seguranca do Concordia.

## Objetivos

- Consolidar modelo de ameacas.
- Registrar controles e mitigacoes.
- Tornar auditoria e revisao de risco rastreaveis.

## Conteudo esperado

- Threat model por componente.
- Politicas de hardening.
- Checklists de revisao.
- Historico de incidentes e correcoes.
