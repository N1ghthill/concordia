# Roadmap

Este roadmap descreve direcao de produto e engenharia do Concordia em alto nivel.

## Fase 0 - Fundacao do projeto

- [x] Estrutura inicial de repositorio open-source.
- [x] Documentacao de governanca, contribuicao e seguranca.
- [x] Diretrizes arquiteturais de alto nivel.

## Fase 1 - Base de plataforma

- [ ] Definir protocolo de comunicacao e eventos.
- [ ] Especificar modelo de autenticacao e autorizacao.
- [ ] Estruturar API publica para clientes e bots.
- [ ] Publicar primeiras decisoes arquiteturais (ADRs).

## Fase 2 - Cliente e servidor

- [ ] Implementar servidor de comunicacao em tempo real.
- [ ] Implementar cliente oficial inicial.
- [ ] Definir contratos de versao e compatibilidade.
- [ ] Publicar ambiente local de desenvolvimento.

## Fase 3 - Ecossistema aberto

- [ ] Lancar SDK inicial para bots e integracoes.
- [ ] Consolidar trilha de extensibilidade.
- [ ] Publicar guias de operacao self-host.
- [ ] Estruturar governanca comunitaria de longo prazo.

## Criterios permanentes

- Privacidade por padrao.
- Transparencia tecnica e documental.
- API aberta e sem lock-in.
- Sustentabilidade do projeto como bem comum digital.
