# scripts

Scripts de suporte para manutenção do repositório e verificações locais.

## Conteúdo

- `scripts/check-structure.sh`: valida a estrutura mínima do projeto.
- `scripts/check-markdown.sh`: valida arquivos Markdown (CRLF, trailing whitespace, arquivo vazio).
- `scripts/branding/`: pipeline utilitário para gerar assets de marca.
