# Branding

Diretório oficial de ativos de identidade visual do Concórdia.

## Conteúdo atual

- `concordia-assets-organizados/`: kit principal consolidado.
- `concordia-export-pack/`: exportação legada inicial.
- `concordia-export-pack-fixed/`: exportação legada revisada.

## Diretriz de manutenção

- `concordia-assets-organizados/` é a referência preferencial para uso atual.
- Pastas legadas devem ser preservadas apenas para rastreabilidade histórica.
- Novos pacotes compactados devem ser direcionados para `assets/archives/`.
