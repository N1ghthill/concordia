# Assets

Este diretório concentra arquivos visuais e materiais de identidade do Concórdia.

## Estrutura

- `assets/branding/`: kits de marca e variações exportadas.
- `assets/source-images/`: imagens-base usadas no processo de criação.
- `assets/archives/`: pacotes compactados de distribuição.

## Política

- Ativos de marca devem ser organizados em pastas nomeadas por contexto.
- Evite duplicação entre formatos finais e arquivos de origem.
- Sempre documente mudanças de naming e versão em `assets/branding/README.md`.
