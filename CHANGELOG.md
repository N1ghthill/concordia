## [0.1.0] - 2026-02-22

### Added

- Estrutura profissional de repositório open-source para o projeto Concórdia.
- Documentos de governança: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`.
- Documentos técnicos de alto nível: `ARCHITECTURE.md`, `ROADMAP.md`.
- Estrutura documental em `docs/` com seções de arquitetura, API e segurança.
- Estrutura de publicação no GitHub em `.github/` com templates de issue e PR.
- Workflow de saúde do repositório para validar estrutura, Markdown e scripts shell.
- Organização de ativos em `assets/` e pipeline utilitário em `scripts/branding/`.
